💾 현재 저장 상태
💡 권장 사용법: 구글시트 연동 + 자동 저장 켜두면 모든 편집이 자동 동기화됩니다. 다른 PC나 휴대폰에서 열 때도 동일한 데이터로 작업 가능.
🔄 작업 순서: ① URL 입력 → ② 연결 테스트 → ③ 시트에서 불러오기 → ④ 자동 저장 ON
